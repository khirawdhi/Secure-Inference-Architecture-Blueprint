# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this project, please report it responsibly.

* Email: [khirawdhi@raykhira.com](mailto:khirawdhi@raykhira.com)
* Subject: **[Security Issue] Secure Inference Architecture Blueprint**

Please include:

* Description of the issue
* Steps to reproduce (if applicable)
* Potential impact
* Suggested mitigation (optional)

---

## Scope

This repository is a **security architecture blueprint**, not a production system.

However, issues related to:

* Incorrect security guidance
* Misleading control recommendations
* Gaps in threat modeling

are considered valid and should be reported.

---

## Response Commitment

* Acknowledge within: **3–5 days**
* Provide initial assessment within: **7–10 days**

---

## Responsible Disclosure

Please do not publicly disclose security issues until they have been reviewed and addressed.

---

## Note

This project is intended for educational and architectural guidance purposes.
It should be adapted and validated before use in production environments.
